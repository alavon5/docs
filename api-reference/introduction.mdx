This reference lists the Bosscart API endpoints available in the current OpenAPI specification.

## Contacts

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/contacts` | List all contacts. |
| `POST` | `/contacts` | Create or update a contact. |
| `GET` | `/contacts/{id}` | Retrieve a contact. |
| `PUT` | `/contacts/{id}` | Update a contact. |
| `DELETE` | `/contacts/{id}` | Delete a contact. |
| `GET` | `/contacts/search-by-email` | Search contact by email. |
